# DEX — Backlink & Link-Building Playbook

> Off-site SEO/authority plan for **dexbyakif.com**. Backlinks are the biggest remaining ranking
> factor for a new site. This is a do-once-then-execute-over-time checklist — none of it needs a live
> session; tick items off as you go.

## Reality check
- The site is **new → ~0 referring domains** (nothing to "analyze" yet; you can't measure a profile
  that doesn't exist). The job is to **build** authority, not audit it.
- **Goal:** a natural, diverse, brand-led link profile that signals trust to Google and gives AI
  engines authoritative sources to cite.
- **Measure it for free:** connect **Google Search Console** (search.google.com/search-console) →
  Links report shows who links to you + which pages. This is your free backlink tracker; revisit
  monthly. (No private data needed from you for the *plan* below.)

## Golden rules
1. **Quality > quantity.** 5 relevant, real links beat 500 spam links.
2. **Anchor text:** mostly **brand** ("DEX", "DEX by Akif Saeed") + naked URLs + natural phrases.
   Avoid stuffing exact-match anchors like "best AI agent agency" — that looks manipulative.
3. **Relevance:** links from AI / automation / business / tech sites are worth far more than random
   directories.
4. **NEVER buy** "1000 backlinks" gigs, PBNs, or link farms — they cause Google penalties. If anyone
   spammy links to you and it grows, disavow via Search Console.

## Tier 0 — Foundation (do first, free, ~1-2 hours)
Establish the brand entity everywhere. These are easy, safe, and feed Google's Knowledge Graph
(pairs with the Organization schema already on the site).
- [ ] **LinkedIn Company Page** (DEX by Akif Saeed) + link the site.
- [ ] **X / Twitter** business profile + site link.
- [ ] **GitHub org/profile** (already `iDevBuddy`) — add site to bio.
- [ ] **Crunchbase** company profile.
- [ ] **Google Business Profile** (if you want local/brand presence).
- [ ] **Founder profiles** — Akif Saeed on LinkedIn / X linking to the site (E-E-A-T signal).

## Tier 1 — Relevant directories (free, high-fit, ~2-3 hours)
AI / agency / SaaS directories. Submit DEX as an AI agent development agency / AI tool.
- [ ] **Product Hunt** — launch DEX (or a free tool/lead magnet). High-authority, real traffic.
- [ ] **There's An AI For That** (theresanaiforthat.com).
- [ ] **Futurepedia**, **AI Tool Hunt**, **TopAI.tools**, **AIxploria** — AI tool directories.
- [ ] **Clutch**, **DesignRush**, **GoodFirms**, **The Manifest** — agency/B2B directories (great for
      client leads too).
- [ ] **BetaList / StartupBase / SaaSHub** — startup directories.
- [ ] **G2 / Capterra** — if/when you have a productized offering.

## Tier 2 — Community & social proof (free, ongoing)
Real participation, not spam. Drop links only where genuinely useful.
- [ ] **Hacker News** — a "Show HN" for a tool/the blog engine, or thoughtful comments.
- [ ] **Reddit** — r/artificial, r/automation, r/smallbusiness, r/Entrepreneur: answer real questions,
      link your relevant blog post when it actually helps. (This also feeds the agent's Reddit
      pain-point stream.)
- [ ] **IndieHackers** — share the build story (the autonomous blog agent is a great story).
- [ ] **Dev.to / Hashnode** — republish select blog posts **with a canonical link back** to dexbyakif.com.
- [ ] **Medium** — the agent can cross-post (publish-medium.mjs exists) with canonical back-links.
- [ ] **LinkedIn / X posts** — share each new blog post; earns shares + occasional links.

## Tier 3 — Content-driven (automatic, compounding)
The blog **earns links over time** — this is why the agent runs. Strengthen it:
- [ ] Publish **linkable assets**: definitive guides, comparisons, original takes, free tools/checklists.
      (The agent's deep-research posts already aim here.)
- [ ] Internal linking between related posts (raises the value of pages that do attract links).
- [ ] Keep content **source-grounded + cited** — authoritative pages attract editorial links and AI
      citations.

## Tier 4 — Outreach (higher effort, highest value)
- [ ] **Guest posts** on AI/automation/business blogs — pitch a unique angle (you have real agent-build
      experience).
- [ ] **HARO / Featured / Help-a-B2B-Writer** — answer journalist requests; earns high-authority links.
- [ ] **Podcasts / interviews** — founder appearances linking back.
- [ ] **Partnerships** — tools you integrate with (CRMs, n8n, etc.); ask for a "built with / partners"
      listing.
- [ ] **Unlinked mentions** — if someone mentions DEX without linking, ask them to add the link.

## Suggested cadence (no live monitoring needed)
| When | Do |
|---|---|
| **Week 1** | All of Tier 0 + 5-8 Tier 1 directories. Connect Search Console. |
| **Weeks 2-4** | Rest of Tier 1; start Tier 2 (1-2 communities you'll actually use). |
| **Ongoing (monthly)** | 1-2 outreach/guest-post pitches; share each blog post; check Search Console Links report; disavow spam if any. |

>
> Hi {name}, I'm Akif from DEX (we build production AI agents for businesses). I read {specific post}
> and had a concrete angle your readers would find useful: {1-line idea grounded in real experience}.
> Happy to write an original, no-fluff piece (with data), or contribute a quote. Either way — great
> work on {their site}. — Akif, dexbyakif.com

## What success looks like
- Month 1: ~10-20 foundational/directory links (brand + naked anchors).
- Month 2-3: first editorial/community links + growing Search Console impressions.
- Ongoing: blog posts accumulate links naturally; domain authority rises; rankings + AI citations
  follow (weeks-to-months — Google-side, automatic).

## Tools (optional)
- **Free:** Google Search Console (links + performance), Ahrefs Webmaster Tools (free for your own
  site), Ubersuggest (limited free).
- **Paid (later, if scaling):** Ahrefs / Semrush for competitor link-gap analysis — feed exports to
  the `backlink-analyzer` skill for a full measured audit.
